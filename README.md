This project is a centralized notification system designed to deliver messages to users through different channels like email, SMS, and push notifications. It ensures that notifications are handled reliably, efficiently, and independently, allowing each channel to scale and evolve without affecting the others. The system is built with flexibility in mind, making it easy to add new notification types or improve existing ones in the future.
